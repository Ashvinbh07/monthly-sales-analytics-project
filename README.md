\# 📊 Monthly Sales Analytics Project



\## 📌 Overview



This project analyzes monthly sales data to extract meaningful insights and trends. The goal is to clean raw data, transform it, and build a structured dataset for analysis and visualization.



\## 🛠️ Tools Used



\* Python (Pandas, NumPy)

\* Jupyter Notebook

\* Power BI



\## 📂 Dataset



\* Raw Data: `sales\_raw\_data.xlsx`

\* Cleaned Data: `sales\_cleaned\_data.csv`



\## 🔧 Data Cleaning Steps



\* Handled missing values

\* Removed duplicates

\* Standardized column names

\* Created structured dataset



\## 📈 Key Insights



\* Monthly sales trends

\* Top-performing categories

\* Revenue distribution



\## 📊 Dashboard



Power BI dashboard created using cleaned dataset.



\## 🚀 How to Run



1\. Open `cleaning.ipynb`

2\. Run all cells

3\. Use cleaned dataset for analysis
